## Install ESMValTool 2.14 with Pixi

```
conda install -c conda-forge pixi
vim ~/.bashrc
which pixi
pixi --version
cd ESMValTool
git status
pixi shell --frozen
```

This will create the default environment ready to be used. The module file is then assembled like so:

```
#%Module 1.0

set prefix /apps/jasmin/community/esmvaltool/ESMValTool/.pixi/

prepend-path     PATH                    $prefix/envs/default/bin:$prefix/envs/default/lib/python3.13/site-packages

setenv ESMFMKFILE /apps/jasmin/community/esmvaltool/ESMValTool/.pixi/envs/default/lib/esmf.mk
source "/apps/jasmin/modulefiles/.modulefiles_include.tcl"
```
